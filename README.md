# CopyFiesBasedText

青を基調としたモダンなUIのファイル検索・コピーツールです。
指定したキーワードがファイル名に含まれるファイルを安全にコピーします。

## Version
**v1.0.0**

## デザインと操作性

-   **Blue Theme**: 目に優しいブルーグレーと鮮やかな青のアクセントカラー。
-   **Interactive**: ボタンのマウスオーバーエフェクトなど、反応の良いUI。
-   **Context Menu**: テキスト入力欄で右クリックメニュー（貼り付け・コピー等）が使用可能。

## 機能

1.  **ファイル名検索**: キーワード（部分一致）でファイルを高速検索。
2.  **安全コピー**: 
    -   読み取り専用アクセスで元ファイルを保護。
    -   同一パスへの上書き防止。
    -   同名ファイルは「新しい場合のみ」更新コピー。
3.  **中断機能**: 処理中にいつでも実行を中止可能。

## 使い方

1.  `python app.py` を実行。
2.  **検索キーワード**を入力（1行に1つ）。
3.  **検索元フォルダ**と**コピー先フォルダ**を指定。
4.  「検索とコピーを開始」ボタンをクリック。

## 動作環境

-   Python 3.8 以上
-   OS: Windows / Mac / Linux
